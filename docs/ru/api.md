# Работа с API

## Создание объектов дрона и камеры

```python
drone = EurusControl(ip, drone_port)
camera = EurusCamera(ip, camera_port)
```

**Входные параметры:**

-   ```ip```: str ip
-   ```port```: int порт
-   ```console_log```: bool true для логирования
-   ```log_file```: str файл для логирования

**Подключение:**

```python
drone.connect()
camera.connect()
```

## Методы дрона 

### connect 

Подключиться к дрону / камере

**Результат:**

- ```True``` - подключение установлено
- ```False``` - ошибка подключения 

**Пример:**

```python
drone.connect()
camera.connect()
```


### disconnect

Отключиться от дрона / камеры

**Результат:**

- ```True``` - отключение выполнено
- ```False``` - ошибка

**Пример:**

```python
drone.connect()
camera.connect()
```

### arm

Включить моторы дрона (ARM)

**Результат:**

- ```True``` — дрон заармлен
- ```False``` — ошибка


### disarm

Выключить моторы дрона (DISARM).

**Результат:**

- ```True``` — дрон дизармлен
- ```False``` — ошибка


### takeoff

Взлёт  дрона на заданную высоту

**Параметры:**

- ```altitude``` - высота взлёта (м)

**Пример:**

``` python
drone.takeoff(1.5)
time.sleep(6)
```

*При каждом взлёте локальные координаты дрона сбрасываются в (0, 0, 0)*


### land

Посадка дрона

**Параметры:**
отсутствуют

*Дрон плавно снижает высоту и выполняет посадку в текущей точке*


### move_to_local_point

Полет к заданной точке в локальной системе координат

**Параметры:**

- ```x``` — координата X (м)
- ```y``` — координата Y (м)
- ```z``` — координата Z (м)
- ```yaw``` — угол поворота (градусы)

**Пример:**

```python
drone.takeoff(1)
time.sleep(5)

drone.move_to_local_point(
    x=2,
    y=0,
    z=1,
    yaw=90
) # Дрон летит на 2 метра вперёд и разворачивается на 90 градусов
```


### move_in_body_frame

Полет к точке относительно текущего положения дрона

**Параметры:**

- ```x``` - движение вперёд (м)
- ```y``` - движение влево (м)
- ```z``` - целевая высота относительно земли (м)
- ```yaw``` - угол поворота (градусы)

**Пример:**

``` python
drone.takeoff(1)
time.sleep(6)

drone.move_in_body_frame(x=4, y=3, z=1)
```


### set_velocity

Установка скоростей движения дрона.

**Параметры:**

- vx — скорость по X (м/с)
- vy — скорость по Y (м/с)
- vz — скорость по Z (м/с)
- yaw_rate — скорость вращения (град/с)

**Пример:**

``` python
drone.set_velocity(
    vx=0.5,
    vy=0,
    vz=0,
    yaw_rate=20
)

time.sleep(3)
drone.set_velocity(0, 0, 0, 0)
```

*Команда действует до следующего вызова set_velocity*


### request_telemetry

Получить текущую телеметрию дрона

**Формат ответа:**

- ```None``` - если телеметрия недоступна
- ```dict``` - при успешном получении данных

**Структура telemetry:**

*state*

- ```connected``` — есть ли соединение
- ```armed``` — состояние моторов
- ```mode``` — текущий режим
- ```system_status``` — состояние системы

*battery*

- ```voltage``` - напряжение (В)
- ```cell_voltage``` -
- ```current``` -
- ```percentage``` — заряд (%)

*local_position*

- ```x, y, z``` - позиция (м)
- ```roll, pitch, yaw``` - углы (градусы)

*velocity*

- ```vx, vy, vz```- скорости (м/с)

*point_reached*

- ```True``` - целевая точка достигнута

**Пример:**

``` python
telemetry = drone.request_telemetry()

if telemetry:
    print("Высота:", telemetry["local_position"]["z"])
    print("Батарея:", telemetry["battery"]["percentage"], "%")
```

### laser_shot

Произвести выстрел из лазерной пушки


**Результат:**

- ```True``` - выстрел выполнен
- ```False``` - ошибка

**Пример:**

``` python
if drone.laser_shot():
    print("Выстрел выполнен")
```

*Команда не блокирует управление полётом*

*Текущий поток блокируется на ~0.5 секунды до получения ответа*


### led-control

Управление светодиодной лентой

**Параметры:**

- ```effect``` - режим свечения

**Поддерживаемые режимы:**

- ```base``` - синий цвет
- ```static``` - статичный цвет
- ```rainbow``` - радуга
- ```komet``` - бегущий эффект
- ```blink``` - мигание

**Дополнительные параметры:**

- ```r, g, b``` - цвет (значения от 0 до 255)
- ```nLED``` - количество светодиодов
- ```brightness``` - яркость

**Результат:**

- ```True``` - команда принята
- ```False``` - ошибка

**Пример:**

``` python
drone.led_control(
    effect="static",
    r=255,
    g=0,
    b=0,
    brightness=200
)
```
