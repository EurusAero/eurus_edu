# Образ Клевера

Платформа Eurus-Edu поддерживает пакет Клевер - комплексное программное обеспечение с открытым исходным кодом, которое предоставляет готовые алгоритмы автономных полётов, инструменты компьютерного зрения и средства симуляции, значительно ускоряя разработку и тестирование дронов для образовательных и исследовательских задач.

**Доступ к пакету открыт и доступен по [ссылке](https://github.com/EurusAero/clover)**

**[Актуальный образ клевера](https://disk.yandex.ru/d/PdhT7mDEIl2g3g)**

**Особенности работы:**

- В настоящий момент не осуществляется поддержка Optical Flow
