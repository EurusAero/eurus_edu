# Лазерная пушка


**Подключение:**

``` python
drone.laser_shot()
```

**Примечание:**

-   Отправляет команду выстрела
-   Не блокирует другие команды управления (можно вызывать параллельно с полётом)
-   Блокирует только текущий поток на ~0.5 секунд до получения ответа о выстреле.


**Выходные параметры:**

-   ```true``` - выстрел успешен
-   ```false``` - выстрелить не получилось


**Пример работы с лазерной пушкой на Python:**

``` python
drone.takeoff(1)
time.sleep(5)

result = drone.laser_shot()

if result:
    print("Выстрел выполнен")
else:
    print("Ошибка выстрела")
```
