# Лазертаг

## 1\. Выстрел лазером

**Использование:**

```python
drone.laser_shot()
```

**Примечание:**

- Отправляет команду выстрела
- Не блокирует другие команды управления (можно вызывать параллельно с полётом)
- Блокирует только текущий поток на ~0.5 секунд до получения ответа о выстреле.

**Выходные параметры:**

- `true` - выстрел успешен
- `false` - выстрелить не получилось

**Пример работы с лазерной пушкой на Python:**

```python
drone.takeoff(1)
time.sleep(5)

result = drone.laser_shot()

if result:
    print("Выстрел выполнен")
else:
    print("Ошибка выстрела")
```

## 2\. Начало игры

**Подключение:**

```python
drone.start_game()
```

**Параметры:**

- `start_game` - флаг запуска игры
- `command_color` - Цвет игрока, определяет принадлежность дрона к команде. Допустимые значения: "red", "blue"

Статус дрона в игре можно проверить через параметр `is_alive`, который находится в сообщении телеметрии `drone.get_telemetry()`

- `true` - дрон жив
- `false` - дрон мёртв
