# Преобразование весов в поддерживаемый тип для RK3588

После того как вы перенесли веса (к примеру best.pt файл) на Orange pi 5 (или любой другой микрокомпьютер с процессором Rockchip 3588), необходимо преобразовать стандартный best.pt в поддерживаемый микрокомпьютером файл.

Для этого можно использовать команду `yolo export`, пример её использования:

```bash
 yolo export model=best.pt format=rknn name=rk3588
```

После этого будет создана папка best_rknn_model, эту папку мы и будем использовать как модель при использовании её в коде.

[Официальная документация Ultralytics по преобразованию модели.](https://docs.ultralytics.com/integrations/rockchip-rknn/)
