# Сборка

### Сборка рамы

1. Соединяем 4 луча с центарльной декой, фиксируя их при помощи винтов M3x12 и закрепляя гайками M3

![center_plate](assets/neural_network_train/assembly/1.png)

2. Устанавливаем моторы на соответствующие отверстия в луче при помощи винтов M3x6. 

![motor](assets/neural_network_train/assembly/2.png)

*Убедитесь, что винты не касаются мотора.*


3. Установите ножки, закрепляя их между собой стойками и винтами M3x6

![leg](assets/neural_network_train/assembly/4.png)

4. На стойки по 6мм устанавливаем регулятор скорости, закрепляем при помощи нейлоновых гаек

5. Установите малую деку на стойки при помощи винтов с потайной головкой

6. Склейте 2–3 слоя двустороннего скотча, приклейте его в центр малой деки и установите полётный контроллер

7. Платформу для Orange Pi 5 закрепляем при помощи стоек 6мм к нижней части центральной деки. Саму плату устанавливаем на стойки по 6мм

![orangepi1](assets/neural_network_train/assembly/5.png)


8. На стойки по 20мм устанавливаем нижнюю платформу, крепим её на винты M3x8

![orangepi2](assets/neural_network_train/assembly/6.png)
