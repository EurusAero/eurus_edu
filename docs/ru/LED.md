# Светодиодная лента


Для работы с лентой подключите ее к питанию +5v – 5v, земле GND – GND и сигнальному порту DIN – GPIO21. Обратитесь к инструкции по сборке для подробностей


**Подключение:**

``` python
led_control(effect)
```

**Входной параметр:**

-   ```effect (string)``` - режим свечения


**Режимы свечения:**

-   ```base``` - постоянное свечение синим
-   ```static``` - постоянное свечение выбранным цветом с выбранными параметрами
-   ```rainbow``` - переливание ленты цветами радуги
-   ```komet``` - бегает по ленте выбранным цветом
-   ```blink``` - мигание выбранным цветом



**Параметры ленты:**

-   ```r, g, b``` - красный, зелёный и синий компоненты сечения (значения изменяются от 0 до 255)
-   ```nLED``` - количество светодиодов
-   ```brightness``` - яркость свечения


**Пример работы с лентой на Python:**

``` python
# Красная подсветка
drone.led_control(
    effect="static",
    r=255,
    g=0,
    b=0,
    brightness=200
)

# Радужный эффект
drone.led_control(effect="rainbow")
```
